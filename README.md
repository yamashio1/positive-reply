# PositiveReply — デプロイ手順

## ファイル構成
```
noise-app/
├── index.html      ← メインアプリ（Tone.js経由でBGM機能も内蔵）
├── bgm.mp3         ← BGM音源（ピアノ・3拍子・ループ用、128kbps）
├── manifest.json   ← PWA（スマホ対応）設定
├── vercel.json     ← Vercelデプロイ設定
├── icon-192.png    ← アプリアイコン（別途用意）
├── icon-512.png    ← アプリアイコン（別途用意）
└── README.md       ← この手順書
```

## BGM機能について

- ヘッダー右上の「BGM オフ」ボタンでON/OFF切替（初期状態はOFF、8言語対応）
- ブラウザの自動再生制限のため、必ずボタンタップで再生開始されます
- ボタン直下に音量スライダーを常時表示（小さく・シンプルに）
- **1曲が終わると、自動で別キーの音源に切り替わります**（重複しない範囲でランダム選択）

### 開発者用の隠しパネル（一般ユーザーには非表示）

ロゴ（PositiveReply）を**1.5秒以内に5回連続クリック**するとパスワード入力欄が出現します。

- パスワードは `index.html` 内の `DEV_PASSWORD` 定数で設定（初期値: `noisereply2026`）
- 正しいパスワードを入力すると、以下の確認用パネルが開きます
  - 現在再生中の曲名
  - 再生位置シークバー
  - 「-10秒」「+10秒」ボタン
  - 「次の曲へ」ボタン（即座に別キーの曲に切替）
- 「パネルを閉じる」ボタンでいつでも非表示にできます
- 一般ユーザーがロゴを誤って連打しても害はなく、パスワードを知らなければ機能には到達できません

**パスワードの変更方法**：`index.html` 内で以下の行を検索し、値を書き換えてください。
```javascript
const DEV_PASSWORD = 'noisereply2026'; // 必要に応じて変更してください
```

### 現在の構成（6曲）

| ファイル | ピッチ |
|---|---|
| bgm_00.mp3  | ±0（オリジナル） |
| bgm_p02.mp3 | +2半音 |
| bgm_p05.mp3 | +5半音 |
| bgm_p09.mp3 | +9半音 |
| bgm_p12.mp3 | +12半音 |
| bgm_m05.mp3 | -5半音 |

1曲が終わると、上記6曲の中から直前と異なる曲がランダムに選ばれ自動再生されます。

### BGM音源の追加方法

`index.html` 内の `BGM_TRACKS` 配列にファイル名を追加するだけです。

```javascript
const BGM_TRACKS = [
  { file: 'bgm_00.mp3', label: 'bgm_00（±0）' },
  { file: 'bgm_p03.mp3', label: 'bgm_p03（+3）' },
  { file: 'bgm_m02.mp3', label: 'bgm_m02（-2）' },
];
```

- ファイルが存在しない場合は自動的にスキップされるので、配列に書いておいて後からファイルを追加してもOK
- 命名規則：`bgm_00`＝オリジナル、`bgm_p03`＝+3半音、`bgm_m02`＝-2半音（p=plus, m=minus）
- 推奨スペック：MP3 96kbps・44.1kHz・3分前後・曲の頭と終わりが自然につながるように編曲

### ファイルサイズの目安（96kbps・3分の場合）

| 曲数 | 合計サイズ |
|---|---|
| 1曲 | 約1.9MB |
| 3曲 | 約5.7MB |
| 18曲 | 約34MB |

18曲構成にする場合は、初回読み込みを軽くするための遅延読み込み実装を別途検討します。

## STEP 1：GitHubにアップロード

1. https://github.com にアクセス、アカウント作成（無料）
2. 「New repository」→ リポジトリ名を `noise-reply` などに設定
3. `index.html` / `manifest.json` / `vercel.json` をアップロード

---

## STEP 2：Vercelでデプロイ

1. https://vercel.com にアクセス、GitHubアカウントでログイン
2. 「Add New Project」→ 作成したGitHubリポジトリを選択
3. 設定はデフォルトのまま「Deploy」をクリック
4. 数分で `https://noise-reply.vercel.app` のようなURLが発行される

---

## STEP 3：独自ドメインの設定（任意）

1. お名前.com / Namecheap などでドメイン取得（例：noisereply.app）
2. Vercelのダッシュボード → 「Domains」→ 取得したドメインを入力
3. ドメイン管理画面でDNS設定（Vercelが表示する値をコピペするだけ）

---

## STEP 4：PWAアイコンの準備

`icon-192.png` と `icon-512.png` を用意してリポジトリに追加。
※ 無料で作成できるツール：https://favicon.io/
